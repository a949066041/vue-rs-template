# Rsbuild Project

## Setup

Install the dependencies:

```bash
pnpm install
```

## Get Started

Start the dev server:

```bash
pnpm dev
```

Build the app for production:

```bash
pnpm build
```

Run lint and type checks:

```bash
pnpm lint
pnpm typecheck
```

Preview the production build locally:

```bash
pnpm preview
```
